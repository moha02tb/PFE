# Admin Pharmacie

React + Vite admin dashboard for PharmacieConnect.

## Commands

```bash
npm install
npm run dev
npm run build
npm run lint
```

## Main Directories

- `src/pages/` - route-level screens
- `src/components/` - shared layout and UI
- `src/context/` - app state providers
- `src/lib/` - API access and utilities
