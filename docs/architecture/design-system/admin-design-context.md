# Design Context - Pharmacy Connect Admin

## Users
**Primary users**: Pharmacy administrators managing operations during business hours (desktop-focused).
**Job to be done**: Quickly access dashboard metrics, manage pharmacies, handle data uploads, view schedules, and manage emergency coverage—all with confidence and efficiency.
**Context**: Office work, high-frequency use across multiple sections.

## Brand Personality
**Three words**: Professional, Modern, Bold  
**Emotional goals**: Confidence, clarity, efficiency, control  
**Core feeling**: A sophisticated platform that respects the user's time and expertise. Not overly playful, but not sterile. Purposeful, not decorative.

## Aesthetic Direction
- **Visual tone**: Modern & Bold with sharp typography, strategic use of color, and intentional hierarchy
- **Theme**: Both light and dark modes with smooth toggle
- **Personality**: Professional but approachable; data-focused but not cluttered
- **Anti-reference**: Avoid corporate blandness, generic admin templates, monotonous gray UIs
- **Key differentiator**: Bold use of the brand primary color, distinctive typography, confident spacing, and deliberate visual hierarchy

## Design Principles
1. **Hierarchy First**: Clear distinction between primary actions, secondary actions, and information through size, weight, and color
2. **Modern Typography**: Distinctive font pairing with intentional weight/size contrasts (not all weights equal)
3. **Bold Colors**: Strategic use of primary brand color (#004AB7 light, #B2C5FF dark) for emphasis; muted neutrals as backdrop
4. **Responsive to Action**: Every interactive element feels responsive and intentional; micro-interactions support understanding
5. **Data-Driven Design**: Tables, dashboards, and lists prioritize scanability and pattern recognition

## Implementation Focus Areas
- Dashboard: Bold hero metrics with clear hierarchy
- Navigation: Clean, modern sidebar with active state emphasis
- Tables: Scannable rows with proper typography weight hierarchy
- Forms: Intentional spacing, clear field hierarchy, confident validation states
- Modals & Drawers: Purposeful animations, clear content hierarchy
- Overall spacing: Varied for rhythm, not monotonous
